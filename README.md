# SPI Interface for LTC1407A ADC and Temperature Sensor using Spartan-3E FPGA

## Project Overview

This project implements an SPI Master Controller in Verilog HDL for interfacing the **LTC1407A dual-channel 14-bit ADC** and a digital temperature sensor using the **Spartan-3E FPGA**.

The controller configures the programmable gain amplifier, initiates ADC conversion, generates SPI clock signals, receives serial ADC data from both channels, and makes the converted digital values available for further processing.

## Working Principle

1. FPGA initializes the SPI interface.
2. Gain amplifier is configured through SPI.
3. ADC conversion is started using the ADC_CONV signal.
4. SPI clock is generated by the FPGA.
5. LTC1407A transmits:
   - 2 Dummy Bits
   - 14-bit Channel A Data
   - 2 Dummy Bits
   - 14-bit Channel B Data
6. FPGA stores:
   - Channel A → adc_data1
   - Channel B → adc_data2
7. Digital output data becomes available for further processing.

## FSM Flow

```
Reset
   │
   ▼
Configure Amplifier
   │
   ▼
Generate ADC_CONV
   │
   ▼
Wait for Conversion
   │
   ▼
Generate SPI Clock
   │
   ▼
Receive Channel-1 Data
   │
   ▼
Skip Dummy Bits
   │
   ▼
Receive Channel-2 Data
   │
   ▼
Store ADC Data
   │
   ▼
Repeat
```

## SPI Data Format

```
Clock Cycles

1-2     Dummy (High-Z)
3-16    Channel-1 (14 Bits)
17-18   Dummy (High-Z)
19-32   Channel-2 (14 Bits)
33-34   End of Transfer
```
